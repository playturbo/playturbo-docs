# 资源组

### 1.什么是资源组 <a href="#rpykv" id="rpykv"></a>

* 资源组是批量制作中的一个重要概念，组内包含若干个在组合生成作品时可替换的资源
* 若有n个加入轨道的资源组，每个资源组内的资源数量分别为m1、m2、m3……、mn，则最终组合视频数量为p=m1\*m2\*m3\*……\*mn，每个组合生成的视频只会挑选每个资源组中的其中一个资源进行视频渲染
* 资源组和普通元件一样可以加入轨道，并成为轨道上编辑的对象

| 资源组        | 资源个数     | 资源个数        |
| ---------- | -------- | ----------- |
| 图片         | 2        | 2           |
| 视频         | 2        | 2           |
| 音乐         | 2        | 2           |
| 文案         | 0        | 3           |
| 最终组合生成视频数量 | =2x2x2=8 | =2x2x2x3=24 |

### 2.资源组分类

* 资源组可分为：视频资源组、音频资源组、图片资源组、文本资源组
* 您可以创建多种类型的资源组，以满足批量视频生产过程中，不同类型资源替换的需求
