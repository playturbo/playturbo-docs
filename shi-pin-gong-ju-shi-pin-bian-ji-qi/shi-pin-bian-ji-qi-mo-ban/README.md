# 视频编辑器模版

当前，视频编辑器模板包含两类：布局模板、视频模板

✨**布局模版 & 视频模版 的区别**

* 布局模版：主要帮助用户解决多尺寸/多语言裂变的问题。 您只需要按照视频整体布局中的素材要求，上传相关素材，即可完成尺寸/语言的裂变。
* 视频模版：主要帮助用户解决空白制作缺少灵感的问题。创意模版中包括多个「场景」，而场景又包含多个「元素」（如片头、片尾、真人剧情等）。您可以通过替换场景，或直接修改模版中的元素来实现创意视频的输出。
