# 布局模版-使用指南

**使用 布局模板 的操作顺序建议：**



**1.上传素材**

根据布局模版结构，将视频、图片、文字等素材上传到相应的坑位中，上传后记得删除原视频模板附带的素材

**2.使用AI消除功能**

可根据实际需求决定是否使用AI消除功能

**3.预览尺寸裂变效果**

所有素材上传完成后，点击右侧尺寸比例预览裂变效果

**4.语言裂变**

点击【下一步】，勾选需要翻译的素材，进行语言裂变。并逐一预览所有语言版本，确认无误

**5.导出**

完成素材组合裂变、视频尺寸裂变、视频语言裂变后，点击【导出】。选择适当的分辨率和码率后，进行导出。

导出任务提交后，可在 菜单栏我的项目>>>导出&下载历史 列表中查看历史任务进展
