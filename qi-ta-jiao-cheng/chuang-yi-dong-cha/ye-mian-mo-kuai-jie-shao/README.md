# 页面模块介绍

当前，创意洞察包含以下三个模块：&#x20;

[tou-fang-shu-ju-zong-lan.md](tou-fang-shu-ju-zong-lan.md "mention")：提供广告账号的总体投放数据分析。并能通过多维度的拆分，对数据进行多维度对比分析，以及时间上的趋势变化分析，帮助用户更好的了解广告账号的**整体**投放情况

[ke-wan-su-cai-fen-xi.md](ke-wan-su-cai-fen-xi.md "mention")：提供试玩素材层级的数据展示，包含投放数据和埋点数据。同样支持多维度拆分，和时间的趋势变化分析。并且提供素材的数据详情，帮助用户更好的了解广告账户下的**可玩素材**的投放和交互情

[ke-wan-su-cai-jiao-hu-fen-xi](ke-wan-su-cai-jiao-hu-fen-xi/ "mention")：主要用于分析用户在试玩素材时的试玩深度、试玩路径及转化、流失情况。此模块包含 [yong-hu-lv-cheng-fen-xi.md](ke-wan-su-cai-jiao-hu-fen-xi/yong-hu-lv-cheng-fen-xi.md "mention")和 [chang-jing-fen-xi.md](ke-wan-su-cai-jiao-hu-fen-xi/chang-jing-fen-xi.md "mention") 两项功能
