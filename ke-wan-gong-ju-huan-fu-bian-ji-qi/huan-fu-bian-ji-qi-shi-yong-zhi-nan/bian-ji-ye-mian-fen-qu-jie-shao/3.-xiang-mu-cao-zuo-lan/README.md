# 3. 项目操作栏

[huan-fu-bian-ji-qi-xia-zai-tui-song.md](huan-fu-bian-ji-qi-xia-zai-tui-song.md "mention")

顶部区域为项目操作栏，支持：

* 返回项目列表
* 恢复默认值：恢复模板初始设置
* 保存项目
* 下载/推送 项目
* 查看项目的导出进度及下载历史
* 复制当前项目、复制当前项目到自由制作、下载模板全部资源
