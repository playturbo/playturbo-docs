# 场景区

<figure><img src="../../../../.gitbook/assets/image (1004).png" alt=""><figcaption><p>场景区</p></figcaption></figure>

场景区用于展示和放置广告项目的流程划分情况，同时也用来储存不同场景的图层内容。可玩广告是由一个个场景拼接而成的。

而场景，我们可以将其理解为PPT，每个场景就是一页PPT，里面储存了不同类型的资产，然后通过简单的「拖拉拽」操作（即事件设置），实现可玩广告制作。



**场景区包括两部分**：

* 全局设置：包含全局场景和全局设置
* 场景：我们也可以称此区域为“普通场景”，展开后有n个场景缩略图
