# 参数设置区

参数设置区是调整场景和图层参数的区域

<figure><img src="../../../../.gitbook/assets/image (204).png" alt=""><figcaption><p>参数设置区</p></figcaption></figure>

<mark style="color:red;">**说明：**</mark>

* 点击左侧**不同的场景**（全局设置/普通场景），右侧显示出的参数内容也不同（例如有全局设置、场景设置、事件、玩法参数）
* 点击左侧**不同的图层**（即不同类型的资产），右侧显示出的参数内容也会不同（例如有外观、动画、事件，不同类型资产的外观参数也有所不同）
* 同时一些我们常见的横竖屏适配问题（适配规则/适配方式）也会在参数设置区这里进行调整

因此，本区域我们将展开对不同的参数一一进行介绍，并备注其所属的场景或图层（"通用"即是点击场景或图层都有该参数）
