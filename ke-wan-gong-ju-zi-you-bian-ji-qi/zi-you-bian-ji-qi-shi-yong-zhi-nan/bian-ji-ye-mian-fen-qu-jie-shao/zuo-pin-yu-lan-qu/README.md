# 作品预览区

<figure><img src="../../../../.gitbook/assets/image (157).png" alt=""><figcaption><p>作品预览区</p></figcaption></figure>

作品预览区由【画布】【场景视图】【横竖屏编辑模式】【多语言预览】【播放场景】【快捷排版】【预览尺寸】【帮助中心】【包体预估】【画布标尺】【参考线】11个小模块组成，此区域是编辑器最核心的操作区域。

（注：【快捷排版】仅在模板自由制作中出现）



## 画布的基本操作逻辑

* 单击选中图层/组
* 组的选中优先级比组内图层元素高
* 要选中组内元素，要事先通过双击激活该组
* 激活组之后，当前组无法被选中
* 可通过框选选择多图层，被隐藏/锁定的图层无法参与框选
